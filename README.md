## 🎯 **Objetivos del Proyecto**

### 📊 **Análisis Principal**
- **Machine Learning aplicado** al mercado tecnológico
- **Análisis específico de Chile** 🇨🇱
- **Comparativa global** vs mercado local
- **Proyecciones 2025-2028** con impacto de IA

### 🤖 **Técnicas Utilizadas**
- **K-Nearest Neighbors (KNN)** para predicción salarial
- **Análisis exploratorio de datos** (EDA)
- **Visualización avanzada** con matplotlib y seaborn
- **Widgets interactivos** con ipywidgets

---

## 📈 **Resultados Principales**

### 🇨🇱 **Mercado Tech Chileno**
- **Salario promedio**: $2.630 USD ($2.367.345 CLP)
- **Premium Santiago**: +22.8% sobre regiones
- **Tecnologías dominantes**: JavaScript (77%), Python (73.3%)
- **Precisión modelo**: R² = 0.759

### 🌍 **Comparativa Global**
- Chile representa **3.5%** del salario promedio global
- Oportunidad de **arbitraje de talento**
- Ventajas competitivas en **nearshoring**

### 🔮 **Proyecciones 2028**
- **Impacto IA**: +30-50% premium para desarrolladores con skills IA
- **Trabajo remoto**: 45% de modalidad total remota
- **Nuevas tecnologías**: TypeScript, AI/ML frameworks
